SEÇENEK 1 (Basit - Önerilen - Hızlı Başlangıç):
- Frontend: HTML, CSS, JavaScript (Vanilla)
- Veri Depolama: localStorage
- Styling: Tailwind CSS CDN veya Bootstrap CDN
- İkonlar: Font Awesome CDN
- Çalıştırma: Live Server (VS Code Extension) veya Python HTTP Server

SEÇENEK 2 (Modern - Daha Gelişmiş):
- Frontend: React + Vite (localhost:5173)
- Veri Depolama: localStorage + Context API
- Styling: Tailwind CSS
- İkonlar: Lucide React
- Çalıştırma: npm run dev

SEÇENEK 3 (Orta Seviye):
- Frontend: Vue.js 3 (localhost:5173)
- Veri Depolama: localStorage + Pinia
- Styling: Tailwind CSS
- İkonlar: Vue Icons
- Çalıştırma: npm run dev

ÖNERİ: Seçenek 1 ile başla - Hızlı, basit ve backend gerektirmiyor!

================================================================================
LOCALHOST ÇALIŞTIRMA YÖNTEMLERİ
================================================================================

YÖNTEM 1 - VS Code Live Server (En Kolay):
1. VS Code'da Live Server extension'ı yükle
2. index.html'e sağ tık > "Open with Live Server"
3. Otomatik olarak http://localhost:5500 açılır

YÖNTEM 2 - Python HTTP Server:
1. Proje klasöründe terminal aç
2. Komut: python -m http.server 8000
3. Tarayıcıda: http://localhost:8000

YÖNTEM 3 - Node.js http-server:
1. Komut: npx http-server -p 8080
2. Tarayıcıda: http://localhost:8080

YÖNTEM 4 - React/Vue (Eğer framework kullanırsan):
1. Komut: npm run dev
2. Tarayıcıda: http://localhost:5173
